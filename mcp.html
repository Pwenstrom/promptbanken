Guide

Hur använder man Promptbanken i praktiken?

Det finns två ganska olika sätt att använda Promptbanken: som vanlig webbsida eller som MCP-server i ett AI-verktyg.

Det enkla sättet

Gå till kommun.promptbanken.se.

Där kan du läsa promptarna, kontrollera hur de är uppbyggda och kopiera dem till en vanlig AI-chatt. Det passar bra om du vill testa snabbt, förstå innehållet eller använda en mall manuellt.

Det mer tekniska sättet

Lägg till Promptbanken som en MCP-server i en AI-klient som stödjer MCP. Då kan klienten hämta promptmallar när du behöver stöd för exempelvis klarspråk, informationsutskick, checklistor, FAQ:er eller beslutsunderlag.

I klienter som stödjer remote MCP via Streamable HTTP kan konfigurationen ofta se ut ungefär så här:

{
  "mcpServers": {
    "promptbanken": {
      "url": "https://mcp.promptbanken.se/mcp"
    }
  }
}

Vissa klienter har ett grafiskt gränssnitt där man lägger till en ny MCP-server och klistrar in adressen. Andra använder en konfigurationsfil. Vissa klienter stödjer bara lokala MCP-servrar. Då behöver man i stället köra Promptbanken MCP lokalt.

Testa gratis

Skapa ett gratis konto om du vill testa Promptbanken med MCP. Med Free får du:

  • tillgång till öppna mallar
  • möjlighet att spara upp till 3 egna mallar
  • 1 personlig MCP-nyckel
  • möjlighet att testa hur Promptbanken fungerar i ett AI-verktyg

Vill du ha fler egna mallar, premium-mallar och fler MCP-nycklar kan du uppgradera till Pro.

Ett sätt att få AI-verktyget att använda Promptbanken

Man kan lägga in en enkel anvisning i sina Custom Instructions, så att AI-verktyget vet när Promptbanken ska användas.

När jag ber om svenska texter, kommunala underlag, mejl, rutiner, checklistor, informationsutskick, FAQ eller beslutsunderlag ska du först kontrollera om Promptbanken har en relevant mall. Om en mall passar ska du använda den och säga vilken mall du valt.

Det gör att man inte behöver skriva “använd Promptbanken” varje gång. Men vill man vara extra tydlig kan man börja chatten med:

  • Promptbanken: gör detta till klarspråk.
  • Använd Promptbanken och skriv ett informationsutskick.
  • Använd Promptbanken. Här är min text. Välj lämplig mall.

Lokal eller remote MCP?

Vill du läsa och kopiera manuellt?Använd kommun.promptbanken.se.
Vill du komma igång snabbt via publik MCP?Använd den publika remote MCP-servern.
Vill du använda egna mallar eller känsligt material?Använd lokal MCP eller kontrollerad intern server.
Vill du dela en kurerad promptbank till flera användare?Remote MCP kan passa, men kräver tydlig styrning.

Den publika remote MCP-servern är https://mcp.promptbanken.se/mcp.

Vilka verktyg stödjer MCP?

Jobbar du i VS Code, Cursor, Windsurf, Codex eller Claude Code kan du ofta använda lokal eller remote MCP, beroende på upplägg.

Microsoft-delen behöver särskiljas:

Microsoft 365 CopilotMCP blir relevant genom connectors eller motsvarande kopplingar. Ta detta med IT.
Copilot StudioHär bygger man en agent som kan använda en MCP-server som verktyg.
VS Code med GitHub CopilotUtvecklaren kan lägga till lokala eller remote MCP-servrar.
Visual Studio med CopilotLiknar VS Code-spåret, men är mer kopplat till Visual Studio och kodnära arbete.

Så lägger du till din MCP-nyckel (Free och Pro)

Alla konton — både Free och Pro — kan skapa en egen MCP-nyckel under Min workspace → MCP-nyckel. Free-konton kan ha 1 aktiv nyckel, Pro-konton upp till 3.

1. Skapa nyckelGå till admin.html → MCP-nyckel → ge den ett namn → "Skapa MCP-nyckel". Nyckeln visas bara en gång, spara den direkt (t.ex. i en lösenordshanterare).
2. Anslut ditt AI-verktygAnvänd något av alternativen nedan beroende på vilket verktyg du kör.
3. Återkalla vid behovTappat bort nyckeln eller misstänker läckage? Återkalla den direkt i admin.html och skapa en ny.

Med nyckeln inkopplad kan AI-verktyget även nå dina egna sparade mallar och, om din workspace har Pro, premium-mallarna — inte bara de öppna standardmallarna.

Vad får MCP-nyckeln komma åt?

Din MCP-nyckel är personlig. Det betyder att den bara ska ge åtkomst till det din plan tillåter.

FreeÖppna mallar och dina egna Free-mallar.
ProÖppna mallar, premium-mallar och dina egna Pro-mallar.

Nyckeln ger inte åtkomst till andra användares privata mallar.

Är du medlem i en delad arbetsyta kan en personlig Pro-nyckel användas mot den delade ytan där du är medlem, men anropet ska vara kontextstyrt — AI-verktyget behöver veta om det ska använda din privata yta eller en viss delad arbetsyta.

Personliga MCP-nycklar används inte för Förvaltning eller Kommun. De nivåerna använder organisationsägda MCP-nycklar.

Koppla din nyckel till ett AI-verktyg

I ett AI-verktyg som stödjer remote MCP lägger du till Promptbankens MCP-server på https://mcp.promptbanken.se/mcp och skickar med din nyckel som ett Bearer-token. En konfiguration kan ofta se ut ungefär så här:

{
  "mcpServers": {
    "promptbanken": {
      "url": "https://mcp.promptbanken.se/mcp",
      "headers": {
        "Authorization": "Bearer DIN_MCP_NYCKEL"
      }
    }
  }
}

Byt ut DIN_MCP_NYCKEL mot din egen nyckel från admin.html. Vissa klienter använder ett annat fält än headers, eller har ett särskilt fält för token/API-nyckel — följ då verktygets egen instruktion och använd samma MCP-adress och samma nyckel.

Säkerhet

Behandla MCP-nyckeln som ett lösenord. Gör inte detta:

  • klistra inte in nyckeln i offentliga chattar
  • lägg inte nyckeln i GitHub
  • dela inte nyckeln med andra
  • skicka inte nyckeln i mejl om det kan undvikas

Om du misstänker att nyckeln har läckt: gå till MCP-nyckel i Promptbanken, återkalla nyckeln och skapa en ny.

Claude (Claude Desktop / Claude Code)

Claude Desktop läser MCP-servrar från en konfigurationsfil. Öppna Claude → Settings → Developer → Edit Config, och lägg till:

Publik server (standardmallar, snabbast att testa):

{
  "mcpServers": {
    "promptbanken": {
      "url": "https://mcp.promptbanken.se/mcp"
    }
  }
}

Vill du även nå dina egna sparade mallar och (med Pro) premium-mallarna via samma publika server, lägg till din nyckel som ett headers-fält enligt avsnittet "Koppla din nyckel till ett AI-verktyg" ovan.

Lokal server (kräver att du klonat repot och kört npm run setup:python i mcp-server/):

{
  "mcpServers": {
    "promptbanken-lokal": {
      "command": "node",
      "args": ["/sokvag/till/promptbanken/mcp-server/scripts/run-mcp.js"],
      "env": {
        "SUPABASE_URL": "https://ditt-projekt.supabase.co",
        "SUPABASE_ANON_KEY": "din-publishable-anon-nyckel",
        "PROMPTBANKEN_MCP_KEY": "din-mcp-nyckel-fran-admin.html"
      }
    }
  }
}

Starta om Claude Desktop efter att du sparat filen. I Claude Code fungerar samma mcpServers-block i projektets .claude-konfiguration.

SUPABASE_URL och SUPABASE_ANON_KEY hittar du i Supabase-projektets inställningar (samma publika nyckel som webbsidan redan använder — inte hemlig). PROMPTBANKEN_MCP_KEY är din egen nyckel från admin.html → MCP-nyckel. Med de tre satta får AI-verktyget även tillgång till verktyget list_pro_templates, som visar Promptbanken Pros 42 premium-mallar — full prompttext om din workspace har aktiv Pro-plan, annars bara en förhandsvisning.

OpenAI / ChatGPT

ChatGPT-appen kör inte lokala MCP-servrar på samma sätt som Claude Desktop. MCP stöds istället via OpenAIs API (Responses API/Agents SDK) eller via ChatGPTs Connectors (Team/Enterprise-planer), och kräver då en remote MCP-adress:

{
  "type": "mcp",
  "server_label": "promptbanken",
  "server_url": "https://mcp.promptbanken.se/mcp"
}

Bygger du en egen integration mot OpenAIs API lägger du in detta som ett verktyg i din anropskod. Har du inte tillgång till Connectors eller API:et kan du fortfarande använda den enkla vägen: kopiera prompttexten direkt från kommun.promptbanken.se eller Promptbanken Pro in i ChatGPT.

Microsoft Copilot Studio

1. Skapa eller öppna en agentI Copilot Studio, gå till din agent (eller skapa en ny).
2. Lägg till verktygUnder "Tools" (Verktyg) → "Add a tool" → "Model Context Protocol".
3. Ange serveradressKlistra in https://mcp.promptbanken.se/mcp som MCP-serverns URL.
4. TestaAnvänd testfönstret i Copilot Studio och be agenten hämta en promptmall, t.ex. klarspråk eller informationsutskick.

Menyerna i Copilot Studio ändras ibland — leta efter "Tools", "Connectors" eller "Model Context Protocol" om stegen ovan inte matchar exakt. Prata med din IT-avdelning innan du kopplar in externa MCP-servrar i en organisationsagent.

Praktisk slutsats

Vill man testa de öppna promptarna som vanlig webbsida är kommun.promptbanken.se enklast.

Vill man testa den publika MCP-servern är mcp.promptbanken.se/mcp rätt väg.

Vill man använda egna mallar, interna texter eller mer verksamhetsnära material bör man hellre köra lokalt eller i en kontrollerad intern miljö.

Free ger 1 MCP-nyckel för enklare test och användning. Pro ger 3 MCP-nycklar och tillgång till din privata Pro-yta och premium-mallar.
