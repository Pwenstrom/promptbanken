Guide

Hur använder man Promptbanken i praktiken?

Det finns två ganska olika sätt att använda Promptbanken: som vanlig webbsida eller som MCP-server i ett AI-verktyg.

Det enkla sättet

Gå till kommun.promptbanken.se.

Där kan du läsa promptarna, kontrollera hur de är uppbyggda och kopiera dem till en vanlig AI-chatt. Det passar bra om du vill testa snabbt, förstå innehållet eller använda en mall manuellt.

Det mer tekniska sättet

Lägg till Promptbanken som en MCP-server i en AI-klient som stödjer MCP. Då kan klienten hämta promptmallar när du behöver stöd för exempelvis klarspråk, informationsutskick, checklistor, FAQ:er eller beslutsunderlag.

I klienter som stödjer remote MCP via HTTP/SSE kan konfigurationen ofta se ut ungefär så här:

{
  "mcpServers": {
    "promptbanken": {
      "url": "https://mcp.promptbanken.se/sse"
    }
  }
}

Vissa klienter har ett grafiskt gränssnitt där man lägger till en ny MCP-server och klistrar in adressen. Andra använder en konfigurationsfil. Vissa klienter stödjer bara lokala MCP-servrar. Då behöver man i stället köra Promptbanken MCP lokalt.

Ett sätt att få AI-verktyget att använda Promptbanken

Man kan lägga in en enkel anvisning i sina Custom Instructions, så att AI-verktyget vet när Promptbanken ska användas.

När jag ber om svenska texter, kommunala underlag, mejl, rutiner, checklistor, informationsutskick, FAQ eller beslutsunderlag ska du först kontrollera om Promptbanken har en relevant mall. Om en mall passar ska du använda den och säga vilken mall du valt.

Det gör att man inte behöver skriva “använd Promptbanken” varje gång. Men vill man vara extra tydlig kan man börja chatten med:

  • Promptbanken: gör detta till klarspråk.
  • Använd Promptbanken och skriv ett informationsutskick.
  • Använd Promptbanken. Här är min text. Välj lämplig mall.

Lokal eller remote MCP?

Vill du läsa och kopiera manuellt?Använd kommun.promptbanken.se.
Vill du testa snabbt via publik MCP?Använd remote MCP och demo-servern.
Vill du använda egna mallar eller känsligt material?Använd lokal MCP eller kontrollerad intern server.
Vill du dela en kurerad promptbank till flera användare?Remote MCP kan passa, men kräver tydlig styrning.

Demo-servern för remote MCP är https://mcp.promptbanken.se/sse.

Vilka verktyg stödjer MCP?

Jobbar du i VS Code, Cursor, Windsurf, Codex eller Claude Code kan du ofta använda lokal eller remote MCP, beroende på upplägg.

Microsoft-delen behöver särskiljas:

Microsoft 365 CopilotMCP blir relevant genom connectors eller motsvarande kopplingar. Ta detta med IT.
Copilot StudioHär bygger man en agent som kan använda en MCP-server som verktyg.
VS Code med GitHub CopilotUtvecklaren kan lägga till lokala eller remote MCP-servrar.
Visual Studio med CopilotLiknar VS Code-spåret, men är mer kopplat till Visual Studio och kodnära arbete.

Praktisk slutsats

Vill man testa de öppna promptarna som vanlig webbsida är kommun.promptbanken.se enklast.

Vill man testa den publika MCP-servern är mcp.promptbanken.se/sse rätt väg.

Vill man använda egna mallar, interna texter eller mer verksamhetsnära material bör man hellre köra lokalt eller i en kontrollerad intern miljö.
