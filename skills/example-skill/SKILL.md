# Example Skill

## Purpose
Help the assistant produce short, structured, and practical responses for a task.

## When to Use
- When the user asks for a clear action plan.
- When the task needs a concise summary.
- When the assistant should avoid unnecessary detail.

## Workflow
1. Identify the user's goal.
2. Extract the minimum useful context.
3. Do the requested work directly.
4. Report the result in a short, structured format.

## Output Format
- Start with the result.
- Add 1-3 bullets only if needed.
- Mention blockers explicitly.

## Example
User: "Can you help me with this task?"

Response:
"Yes. I will first inspect the relevant files, then make the smallest safe change, and finally verify the result."
